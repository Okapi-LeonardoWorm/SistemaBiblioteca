# Documentacao do Sistema Biblioteca

Esta pasta centraliza toda a documentacao do projeto em duas categorias:

- Interna: documentacao tecnica para desenvolvimento, manutencao e operacao.
- Externa: documentacao de uso do sistema para perfis funcionais.

## Estrutura

- internas/
- externas/

## Escopo inicial

A trilha interna cobre backend e frontend:

- Backend em `docs/internas/backend` com foco em arquitetura, modelos, rotas, servicos, utilitarios e troubleshooting.
- Frontend em `docs/internas/frontend` com foco em componentes reutilizaveis, contratos de integracao com API e padroes de UX.
