# Operacao Diaria do Sistema

Publico alvo: Administrador.

Este guia externo descreve como executar tarefas do dia a dia sem detalhes tecnicos de backend.

## Como usar esta documentacao

Cada caso de uso segue o mesmo formato:

1. Objetivo
2. Quando usar
3. Pre-requisitos
4. Passo a passo
5. Como preencher os campos
6. Erros comuns e como resolver
7. Resultado esperado

## Matriz de fluxos

### Acesso

- Realizar login e iniciar navegacao
- Usar o dashboard administrativo (perfil admin)

### Usuarios

- Cadastrar usuario
- Editar usuario
- Consultar historico de emprestimos no usuario
- Excluir usuario (exclusao logica)
- Reativar usuario
- Importar usuarios em massa

### Livros

- Cadastrar livro
- Editar livro
- Consultar historico de emprestimos no livro
- Excluir livro (exclusao logica)
- Reativar livro

### Emprestimos

- Cadastrar emprestimo
- Editar emprestimo
- Cancelar emprestimo (quando permitido)
- Informar retorno (concluido ou perdido)

### Tags

- Cadastrar tag
- Cadastrar varias tags de uma vez
- Editar tag
- Consultar livros por tag
- Excluir tag (exclusao logica)
- Recuperacao de tag (orientacao operacional)

### Backup

- Configurar credenciais Google Drive
- Conectar e desconectar integracao Google
- Executar backup manual
- Configurar agendamentos
- Monitorar fila e tentativas de upload
- Resolver falhas e recuperar pasta invalida
