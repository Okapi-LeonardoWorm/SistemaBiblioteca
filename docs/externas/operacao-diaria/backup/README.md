# Backup - Operacao Diaria

Publico alvo: administrador do sistema (nao tecnico).

## Objetivo

Orientar configuracao e operacao do backup de ponta a ponta:

1. Configurar Google Drive
2. Conectar conta
3. Executar backup
4. Monitorar fila de upload
5. Resolver falhas

## Ordem recomendada

1. google-drive-setup.md
2. configuracoes-e-preenchimento.md
3. agendamentos.md
4. monitoramento-da-fila.md
5. recovery-mode.md
6. troubleshooting.md

## Resultado esperado

Ao seguir os guias, o operador consegue:

- habilitar integracao com Google Drive
- executar backup manual e agendado
- entender tentativas e erros de upload
- validar que o arquivo foi enviado com sucesso
