# Livros por Tag

## Objetivo

Consultar os livros vinculados a uma tag e abrir rapidamente a edicao de um livro especifico.

## Quando usar

- Para validar onde uma tag esta sendo utilizada.
- Para localizar livros por classificacao sem navegar pela listagem geral de livros.

## Pre-requisitos

- Estar logado.
- Abrir uma tag existente em modo edicao.

## Passo a passo

1. Acesse Gerenciar Tags.
2. Abra a tag desejada.
3. No modal, selecione a aba Livros.
4. Verifique o dashboard:
   - Quantidade de livros
5. Use a busca para filtrar por:
   - Nome do livro
   - Nome do autor
6. Clique em uma linha para abrir o modal de edicao do livro.

## Como interpretar a listagem

- Livro: titulo do livro associado a tag.
- Autor: autor principal do livro.

## Erros comuns e como resolver

1. Nenhum livro aparece para a tag:
   - confira se a tag esta realmente associada a livros ativos.
2. Busca nao encontra o livro esperado:
   - tente termos parciais do nome do livro e do autor.
3. Texto aparece com ...:
   - passe o mouse sobre o campo para visualizar o texto completo.

## Resultado esperado

A listagem mostra os livros vinculados a tag, o dashboard informa o total e o clique em um item abre a edicao do livro correspondente.
