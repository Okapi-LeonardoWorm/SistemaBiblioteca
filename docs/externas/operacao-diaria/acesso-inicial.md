# Acesso Inicial ao Sistema

## Objetivo

Orientar o primeiro acesso e a navegacao inicial ate as telas de operacao diaria.

## Quando usar

- Primeiro acesso do dia.
- Retorno ao sistema apos encerramento de sessao.

## Pre-requisitos

- Possuir codigo de identificacao e senha validos.

## Passo a passo

1. Abra a tela de Login.
2. Preencha Usuario e Senha.
3. Clique em Login.
4. Acesse o menu inicial.
5. Entre no modulo desejado (livros, usuarios, emprestimos ou tags).

## Como preencher os campos

- Usuario: informe seu codigo de identificacao.
- Senha: informe sua senha de acesso.

## Erros comuns e como resolver

- Usuario ou senha invalidos: revisar dados digitados e tentar novamente.
- Nao consigo entrar no dashboard: apenas administradores acessam painel administrativo; use o menu operacional quando aplicavel.

## Resultado esperado

Sessao iniciada e navegacao liberada para as funcionalidades permitidas ao perfil.
